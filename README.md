# Amazon Product Review Insights 

## Project Overview
This project presents an interactive **Power BI dashboard** analyzing Amazon product reviews to understand customer behavior, rating trends, and product category performance.  
The dashboard focuses on how **ratings and review volume vary across product types** and provides high-level KPIs for business decision-making.

## Key KPIs
- **Total Products:** 1,351  
- **Average Rating:** 4.10  
- **Total Reviews:** 26.77M  

## Dashboard Insights
- Identifies **most reviewed product categories** on Amazon  
- Compares **average customer ratings by product type**  
- Analyzes the relationship between **ratings and total reviews** using a bubble chart  
- Shows **rating distribution** using a histogram  
- Provides a **summary table** with conditional formatting (data bars) for quick comparison

## Visualizations Included
- KPI Cards (Product Count, Avg Rating, Total Reviews)
- Horizontal bar charts
- Bubble chart (Ratings vs Reviews)
- Histogram (Rating Distribution)
- Table with conditional formatting (Data Bars)

## Tools & Technologies

## Key Business Insights
- Most Amazon products maintain high customer ratings above 4.0.
- Categories with higher review volume are not always the highest-rated.
- Review distribution is heavily concentrated between 4.0–4.5 stars.
- Popular categories indicate strong customer engagement and demand.
